# zhihu-daily-vue

> 使用 vue-cli + vuex + vue-router + vue-carbon 实现的知乎日报

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:28080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
